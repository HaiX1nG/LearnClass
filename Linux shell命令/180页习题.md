# 实践习题

## 1.创建一个脚本,运行该脚本时,显示:你目前的身份(用`whoami`); 你的目前所在的目录(用`pwd`).

```powershell
[root@Server02 ~]# vim sh20.sh
```

```sh
#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin  export PATH

echo "你目前的身份：$(whoami)"
echo "你的目前所在的目录：$(pwd)"
```

```powershell
[root@Server02 ~]# sh sh20.sh
你目前的身份：root
你的目前所在的目录：/root
```
